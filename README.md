## Run

To be able to run, you will need to create a user.
To make your work easier, put these in the parameters:

|field     | value   |
|----------|---------|
|username  |user     |
|name      |user     |
|email     |user@t.c |
|password  |1        |

### Courier

To be able to see the courier interface, you need to login with this account: 

|field     | value   |
|----------|---------|
|username  |courier  |
|password  |1        |
